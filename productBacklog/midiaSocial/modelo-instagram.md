# Modelo de perfil Instagram — ONG (exemplo)

ONG ficticia de referencia: **Instituto Semente do Sudoeste**.

## Configuracao do perfil (conta profissional)

| Campo | Conteudo modelo |
|---|---|
| Tipo de conta | Profissional -> Organizacao sem fins lucrativos |
| @usuario | @institutosemente |
| Nome | Instituto Semente | Educacao |
| Categoria | Organizacao sem fins lucrativos |
| Bio (150 caracteres) | ver abaixo |
| Link | https://linktr.ee/institutosemente (ou site direto) |
| Botao de acao | Doar (via parceiro) / E-mail / WhatsApp |

## Bio modelo (ate 150 caracteres)

```
🌱 Educacao que transforma o Sudoeste do PR
📍 Francisco Beltrao
✅ +1.200 criancas atendidas
👇 Doe e acompanhe nosso impacto
```

## Destaques (highlights) iniciais

1. **Quem somos** — missao, equipe, sede.
2. **Impacto** — numeros, depoimentos.
3. **Projetos** — o que esta ativo agora.
4. **Transparencia** — prestacao de contas, relatorios.
5. **Como ajudar** — doar, ser voluntario, empresa parceira.

## Grade inicial (9 posts para lancamento)

| # | Tipo | Conteudo |
|---|---|---|
| 1 | Apresentacao | Missao + foto da equipe |
| 2 | Impacto | Card com numero (ex: 1.200 criancas) |
| 3 | Storytelling | Historia de uma crianca atendida (com autorizacao) |
| 4 | Bastidor | Dia a dia do projeto |
| 5 | Transparencia | "Para onde vai sua doacao" (grafico simples) |
| 6 | Parceiro | Agradecimento a empresa/financiador |
| 7 | Chamada empresa | "Transforme imposto em impacto" (lei de incentivo) |
| 8 | Depoimento | Fala de familia/voluntario |
| 9 | CTA doacao | "Doe agora" com link |

## Boas praticas

- Sempre pedir autorizacao de imagem (LGPD) para fotos de criancas/beneficiarios.
- Usar hashtags locais + tematicas: #FranciscoBeltrao #TerceiroSetor #ODS4 #Educacao #ImpactoSocial.
- Postar 2-3x por semana no minimo.
- Marcar parceiros e financiadores nos posts de agradecimento.

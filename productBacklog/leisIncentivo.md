## Como usar este catalogo

- Cada linha "elegivel" vira um **mecanismo** disponivel no match e na calculadora.
- Campo "mecanismo legal" do projeto referencia este catalogo.
- Ao entrar num novo estado/municipio, mapear os incentivos locais antes de onboarding.

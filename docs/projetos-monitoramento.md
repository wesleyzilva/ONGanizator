# Projetos & Monitoramento — Transcrição

Transcrição da estrutura de cadastro de projetos, planejamento orçamentário, monitoramento e prestação de contas digital.

## Dashboard de Projetos — Menu

- Criar Projeto
- Editar Projeto
- Atualizar Projeto
- Relatórios
- Evidências
- Prestação de Contas

## Estrutura do cadastro de projetos

Identificação
- Nome do projeto
- Resumo executivo
- Período de execução
- Área temática

ODS relacionadas (seleção múltipla)

Planejamento lógico
- Objetivo Geral
  - Objetivos específicos
    - Metas
      - Atividades
        - Itens de despesa

Para cada atividade:
- Descrição
- Quantidade
- Unidade
- Valor unitário
- Valor total
- Categoria (RH, material, equipamentos, serviços terceirizados, comunicação, transporte, alimentação, infraestrutura, tecnologia, administrativo)

## Acompanhamento

- Cronograma
- Execução financeira
- Execução física
- Indicadores
- Beneficiários atendidos

## Evidências

- Upload contínuo de fotos, vídeos, relatórios, depoimentos

## Barra de completude do projeto (exemplo)

- Dados Gerais 100%
- ODS 100%
- Metas 80%
- Atividades 70%
- Orçamento 60%
- Indicadores 50%
- Documentos 40%
- Projeto Completo 71%

## Prestação de Contas Digital

Para cada despesa:
- Nota fiscal, recibo, contrato, comprovante PIX, evidência fotográfica, relatório de execução

Geração automática de PDFs (executivo, técnico, resumido) e biblioteca única de evidências organizada por organização, projeto, meta e atividade.
